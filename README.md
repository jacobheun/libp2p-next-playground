# Libp2p Next Playground

This is a testing playground for comparing the js-libp2p async refactor (beta) to the current (latest) version of js-libp2p.

## Running Examples

**Beta**
```sh
$ node beta.js
```

**Latest**
```sh
$ node beta.js
```

**Beta Dialing Latest**
```sh
$ node beta-to-latest.js
```

**Latest Dialing Beta**
```sh
$ node beta-to-latest.js
```
